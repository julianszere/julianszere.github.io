Guitar Simulation

github.com/julianszere/strings

Each string of a guitar is a continuous medium clamped at both ends. The physics is captured by the damped wave equation (leaving out the interaction with the guitar body), and numerical integration of that equation produces both the animation of the vibrating string and the actual audio you would hear. Six strings, each with its own tension and length tuned to match the pitch of a standard guitar.

Smoke on the Water, simulated string by string.

The Physical Model

The transverse displacement $u(x,t)$ of a string of length $L$ evolves according damped wave equation:

$$\frac{\partial^2 u}{\partial t^2} + 2\gamma\,\frac{\partial u}{\partial t} = c^2\,\frac{\partial^2 u}{\partial x^2} - \lambda^2\,\frac{\partial^4 u}{\partial x^4}$$

The first two terms are the standard damped wave equation: $c$ is the wave speed set by string tension and linear density, and $\gamma$ controls how fast the vibration decays. The $\partial^4 u / \partial x^4$ term is a stiffness correction: real strings resist bending slightly, which makes higher harmonics land a little sharp relative to integer multiples of the fundamental. That inharmonicity is part of what gives plucked strings their characteristic timbre.

Boundary conditions pin both ends:

$$u(0,t) = u(L,t) = 0$$

The initial condition is a Gaussian bump near the quarter-length point of the string, mimicking a pluck.

Pitch and Frets

Each open string has a fundamental frequency $f_0$ determined by its wave speed $c$ and scale length $L$:

$$f_0 = \frac{c}{2L}$$

Pressing a fret shortens the vibrating portion of the string to a new length $L' < L$, raising the pitch proportionally. The six open strings are tuned to E, A, D, G, B, e (standard guitar tuning), and each fret position is computed from the standard fret-spacing formula derived from equal temperament. Chords are built by specifying a fret number per string and running the simulation independently for each.

Audio Synthesis

Once the PDE is solved numerically, the displacement field $u(x,t)$ is a grid of values. To extract what a microphone would pick up, the solution is projected onto the first ten normal modes of the string:

$$a_n(t) = \sum_x u(x,t)\,\sin\!\left(\frac{n\pi x}{L}\right)$$

Summing these harmonic amplitudes over time gives a scalar signal that is downsampled to 44 kHz and written to a WAV file. The chord audio is obtained by adding the six string signals together before writing.

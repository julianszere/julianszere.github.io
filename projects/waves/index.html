PDE Image SolverIntegrador de imágenes

github.com/julianszere/imagesolver

A static image is a two-dimensional array of values, which is also exactly what a PDE solver expects as an initial condition. Here, we load an image, interpret each pixel's intensity as the value of a field $u(x, y)$, and then let the field evolve forward in time according to a chosen equation. The result is an animation that shows how the physics of that equation would deform, spread, or oscillate the image over time.

Wave EquationEcuación de onda

The 2D wave equation propagates disturbances at a finite speed $c$:

$$\frac{\partial^2 u}{\partial t^2} = c^2 \nabla^2 u$$

Each RGB channel is evolved independently. The effect is genuinely wave-like, with peaks and troughs radiating outward from every bright edge. As the simulation runs, the channels gradually dephase and fine interference ripples accumulate across the field, eventually dominating the image.

Heat EquationEcuación de calor

The heat (diffusion) equation, as expected, spreads values toward their spatial average at a rate controlled by the diffusion coefficient $D$:

$$\frac{\partial u}{\partial t} = D\,\nabla^2 u$$

Sharp edges blur out immediately and the image progressively melts into uniform grey. There is no oscillation and no wavefront, just an irreversible smoothing that erases all structure.

Schrödinger EquationEcuación de Schrödinger

The free-particle Schrödinger equation evolves a complex wavefunction $\psi$:

$$i\hbar\,\frac{\partial \psi}{\partial t} = -\frac{\hbar^2}{2m}\,\nabla^2\psi$$

The image initialises the amplitude $|\psi|$ of the wavefunction as the square root of each pixel's brightness, so that the probability density $|\psi|^2$ matches the original image at $t = 0$.

To actually solve this equation, the Laplacian $\nabla^2$ is discretised on the pixel grid using finite differences, building a sparse Hamiltonian $H$ of size $(N_x N_y) \times (N_x N_y)$. The formal solution to the equation is $\psi(t) = e^{-iHt}\psi_0$, and each frame is computed by applying one step of this matrix exponential:

$$\psi_{n+1} = e^{-iH\,\Delta t}\,\psi_n$$

The matrix exponential is evaluated with a Krylov subspace method to apply $e^{-iH\Delta t}$ to the state vector without ever forming the full dense matrix. This is exact: no truncation error from time-stepping. Each frame then displays $|\psi|^2$, the probability density.

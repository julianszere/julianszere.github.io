Lagrange Solver

github.com/julianszere/lagrangesolver

Deriving equations of motion by hand gets tedious quickly — and error-prone for anything beyond a double pendulum. This tool takes the geometry of a mechanical system as input and produces the full equations of motion automatically, then integrates them and renders an animation. The only inputs required are the position of each particle in Cartesian coordinates, expressed in terms of generalized coordinates, and the potential energy.

Triple pendulum simulated from automatically derived equations of motion.

How It Works

The user provides each particle's position vector as a string (for example, "l1*cos(phi1)*xhat + l1*sin(phi1)*yhat") along with the potential energy and parameter values. SymPy parses these strings, identifies the generalized coordinates automatically (any symbol that is not a declared parameter), and promotes them to time-dependent functions $q^u(t)$.

From the positions $\mathbf{r}_i(q^1, \dots, q^n)$, the kinetic energy is assembled via the mass matrix:

$$T = \frac{1}{2}\sum_{u,v} M_{uv}\,\dot{q}^u\dot{q}^v, \qquad M_{uv} = \sum_i m_i\,\frac{\partial \mathbf{r}_i}{\partial q^u}\cdot\frac{\partial \mathbf{r}_i}{\partial q^v}$$

The Lagrangian $\mathcal{L} = T - V$ is then formed symbolically, and SymPy applies the Euler-Lagrange operator to each coordinate:

$$\frac{d}{dt}\frac{\partial \mathcal{L}}{\partial \dot{q}^u} - \frac{\partial \mathcal{L}}{\partial q^u} = 0 \qquad u = 1, \dots, n$$

Because $T$ is quadratic in the velocities, the $n$ equations are linear in the accelerations $\ddot{q}^u$. The code solves this linear system symbolically to get closed-form expressions for each $\ddot{q}^u$, lambdifies them into NumPy functions, and feeds the resulting first-order system to scipy.integrate.odeint. The trajectory is then mapped back to Cartesian coordinates for the animation.

Triple Pendulum

Three masses connected in series, each suspended from the one above. The generalized coordinates are the angles $\phi_1$, $\phi_2$, $\phi_3$ measured from vertical. The position vectors fed to the solver are:

$$\left\{\begin{array}{l} \mathbf{r}_1 = \begin{pmatrix}l_1\cos\phi_1\\l_1\sin\phi_1\end{pmatrix}\\[0.8em] \mathbf{r}_2 = \begin{pmatrix}l_1\cos\phi_1+l_2\cos\phi_2\\l_1\sin\phi_1+l_2\sin\phi_2\end{pmatrix}\\[0.8em] \mathbf{r}_3 = \begin{pmatrix}l_1\cos\phi_1+l_2\cos\phi_2+l_3\cos\phi_3\\l_1\sin\phi_1+l_2\sin\phi_2+l_3\sin\phi_3\end{pmatrix} \end{array}\right.$$

The potential energy is:

$$V = -g\bigl[(m_1+m_2+m_3)l_1\cos\phi_1 + (m_2+m_3)l_2\cos\phi_2 + m_3 l_3\cos\phi_3\bigr]$$

From these, the solver assembles the mass matrix and computes the Lagrangian $\mathcal{L} = T - V$. Simplifying with $\sin^2\!+\cos^2\!=1$ and $\cos(\phi_i-\phi_j)=\cos\phi_i\cos\phi_j+\sin\phi_i\sin\phi_j$:

$$\mathcal{L} = \tfrac{1}{2}(m_1{+}m_2{+}m_3)l_1^2\dot\phi_1^2 + \tfrac{1}{2}(m_2{+}m_3)l_2^2\dot\phi_2^2 + \tfrac{1}{2}m_3 l_3^2\dot\phi_3^2$$

$$+\;(m_2{+}m_3)l_1 l_2\cos(\phi_1{-}\phi_2)\dot\phi_1\dot\phi_2$$

$$+\;m_3 l_1 l_3\cos(\phi_1{-}\phi_3)\dot\phi_1\dot\phi_3 + m_3 l_2 l_3\cos(\phi_2{-}\phi_3)\dot\phi_2\dot\phi_3$$

$$+\;g\bigl[(m_1{+}m_2{+}m_3)l_1\cos\phi_1 + (m_2{+}m_3)l_2\cos\phi_2 + m_3 l_3\cos\phi_3\bigr]$$

Applying the Euler-Lagrange operator to each of the three coordinates yields a coupled linear system in $\ddot\phi_1$, $\ddot\phi_2$, $\ddot\phi_3$:

$$\left\{\begin{gathered} (m_1{+}m_2{+}m_3)l_1\ddot\phi_1 + (m_2{+}m_3)l_2\cos(\phi_1{-}\phi_2)\ddot\phi_2 + m_3 l_3\cos(\phi_1{-}\phi_3)\ddot\phi_3\\ +\;(m_2{+}m_3)l_2\sin(\phi_1{-}\phi_2)\dot\phi_2^2 + m_3 l_3\sin(\phi_1{-}\phi_3)\dot\phi_3^2 + (m_1{+}m_2{+}m_3)g\sin\phi_1 = 0\\[0.6em] (m_2{+}m_3)l_1\cos(\phi_1{-}\phi_2)\ddot\phi_1 + (m_2{+}m_3)l_2\ddot\phi_2 + m_3 l_3\cos(\phi_2{-}\phi_3)\ddot\phi_3\\ -\;(m_2{+}m_3)l_1\sin(\phi_1{-}\phi_2)\dot\phi_1^2 + m_3 l_3\sin(\phi_2{-}\phi_3)\dot\phi_3^2 + (m_2{+}m_3)g\sin\phi_2 = 0\\[0.6em] l_1\cos(\phi_1{-}\phi_3)\ddot\phi_1 + l_2\cos(\phi_2{-}\phi_3)\ddot\phi_2 + l_3\ddot\phi_3\\ -\;l_1\sin(\phi_1{-}\phi_3)\dot\phi_1^2 - l_2\sin(\phi_2{-}\phi_3)\dot\phi_2^2 + g\sin\phi_3 = 0 \end{gathered}\right.$$
